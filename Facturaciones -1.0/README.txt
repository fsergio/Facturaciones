Facturaciones-1.0

########Abrir aplicacion por primera vez, cargar ubicacion base de datos #######
Normalmente para windows su ubicacion es: 
	C:\Program Files\Facturaciones -1.0\BASE_25417V3.0.GDB

- Si tiene problemas, Verificar los siguientes parametros en archivo de configuracion firebird:
	WireCrypt = Enabled 
	encoding = NONE
Normalmente para windows su ubicacion es:
	C:\Program Files\Firebird\Firebird_3_0\firebird.conf(modificar como administrador)

- para modificar parametros de conexion de app, abrir cfg.properties ubicado en carpeta raiz

- Herramientas:
	Java Version 1.8.0_131
	Firebird Version 3.0
	jaybird-full-3.0.1(incluido)
